⚙️ 首次配置 - 运营后台设置向导

💡 提示：只需配置一次！配置完成后，刷新页面即可直接使用。
1部署云函数（只需一次）
在微信开发者工具中，右键点击以下云函数文件夹 → 「上传并部署：云端安装」
  • adminGetOrders - 获取订单列表
  • adminShip - 发货（发邮件）
  • uploadProcessedPhoto - 上传照片
2开启云接入（创建 HTTP 访问路径）
  1. 在微信开发者工具中，点击顶部菜单 云开发
  2. 进入「云开发控制台」
  3. 点击左侧菜单 更多 → 云接入
  4. 点击「添加 HTTP 访问路径」
  5. 路径填写 /admin（或其他你喜欢的名称）
  6. 选择要接入的云函数：adminGetOrders
  7. 点击确定，等待部署完成
📍 路径：云开发控制台 → 更多 → 云接入 → 添加 HTTP 访问路径
3复制云接入 URL
部署完成后，在云接入列表中复制完整的 URL，看起来像这样： 
https://cloud1-xxxx.tcloudbaseapp.com/admin

注意：如果 URL 末尾没有路径（如 /admin），你需要手动加上。

4填入 URL 并测试
